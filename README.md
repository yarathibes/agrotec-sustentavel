# Agro Forte - Futuro Sustentável

Este é um site interativo com foco em **produção agrícola sustentável** e **meio ambiente**.

## Estrutura do projeto

- `index.html` — Página principal
- `style.css` — Estilo do site
- `script.js` — Interatividade com JavaScript
- `README.md` — Documentação do projeto

## Funcionalidades

- Botão "Saiba mais" para expandir informações
- Cards com projetos sustentáveis
- Formulário de contato com validação simples
- Design responsivo e moderno

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
